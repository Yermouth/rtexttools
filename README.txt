INSTALLATION
============
To build and install RTextTools, run the following commands while in the rtexttools hg repository:

R CMD REMOVE RTextTools
R CMD BUILD RTextTools
R CMD INSTALL RTextTools_1.3.2.tar.gz

SOURCE CODE
============

To modify R sourcecode, go to the RTextTools folder, and modify files within the R directory. You must have no more than one function per file.
